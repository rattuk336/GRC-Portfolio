## 📚 What I Learned

### Project 1: Risk Register
- Risk is **Likelihood × Impact** - you can't just guess, you need a scoring system
- **Inherent vs Residual risk** shows the value of controls
- A risk register is a **living document**, not a one-time exercise

### Project 2: Data Classification Policy
- Not all data is equal - **PUBLIC, INTERNAL, CONFIDENTIAL, RESTRICTED**
- A policy is useless if employees don't understand it (clear examples matter)
- **Balancing security and usability** is the real challenge

### Project 3: Vendor Risk Assessment
- Your security is only as strong as your **weakest vendor** (Target breach example)
- Vendor assessment is **continuous monitoring**, not a one-time check
- Even "good" vendors have gaps that need **management attention**
